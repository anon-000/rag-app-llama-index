## { RAG app based on pdf base x Gemini }
-------------------
# How to run it ?
```

* git clone
* create a virtual environment
* pip install -r requirements.py
* create a .env file and store your gemini key inside "GOOGLE_API_KEY" variable
* then run using python app.py

```
